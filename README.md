## 🧭 What I Care About

- **Frontend architecture that teams can own independently** — monorepos, shared libraries, design systems that don't become bottlenecks
- **Performance as a habit, not a sprint** — Core Web Vitals in CI, real user monitoring, catching regressions before users do
- **AI-augmented engineering** — MCP servers, design-to-code automation, using AI where it multiplies output not replaces thinking
- **Quality that ships** — Playwright E2E on every PR, error budgets, rollback plans before deployment not after
